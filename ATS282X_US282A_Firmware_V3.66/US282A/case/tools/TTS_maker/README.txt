1、准备音频资源，格式为MP3，采样率为16kbps，单声道；并且不能包含ID3数据，即纯粹的MP3帧数据；
2、音频资源列表从ID为0开始填写对应的文件名字，ID表查看 common_func.h ；项目无须用到的音频资源，
   如果不想修改 common_func.h 中的ID表，那么可以将对应项改为NULL即可；名字后面不能有其他字符；
3、双击 tts_maker.exe 即可生成 tts 资源库 tts_rc.lib ；

注意：生成的音频资源文件不要播放后再打包为 tts_rc.lib ，因为有些播放器播放时会自动添加ID3标签内容，
      这样资源文件开头和结尾可能包含非MP3帧的数据，解码时会出错。所以，如果要试播，请拷贝到另外目
      录进行。
